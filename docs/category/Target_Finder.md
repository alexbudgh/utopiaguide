## Target Finder

Utopia offers 2 different target finders. There is an in-game target finder in the War Room, or you can utilize the target finder in the Intel Site.

## In-Game

- To utilize the In-Game target finder, Enter War Room and select the "potential targets" tab. This will display the In-Game target finder.
  - Ranges:
    - 70%-130%
    - 80%-120%
    - 90%-110%
  - Exclude in-war
  - Race Selection
  - Display Results:
    - Province
    - Race
    - Networth
    - Land
    - NW/Acre
    - KD NW
    - Gains(Bad, Fair, Good, Great, Excellent)
      - Estimated gains is solely based on province Land and Networth, as well as the average networth of provinces in the target Kingdom.
