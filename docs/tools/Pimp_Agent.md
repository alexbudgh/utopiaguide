# Pimp Agent

**Developer:** Molesquirrel & Chid

Pimp Agent is an add-on for [Mozilla Firefox](../misc/Mozilla_Firefox.md) that links directly to your kingdom's [UtopiaPimp](UtopiaPimp.md) account. It builds itself into the Utopia page so you can send data without any copy/pasting or extra browser windows.

> **Note:** With UtopiaPimp closing down, [UtopiaPimp V2](UtopiaPimp_V2.md) is in development and may supersede Pimp Agent.

## Requirements

- A UtopiaPimp account
- Mozilla Firefox

## Features

- Send all ops (attack, thief, mage) straight to UtopiaPimp with a single click
- Send all self-intelligence and enemy-intelligence, including kingdom pages and CE/kingdom news
- Automatically send self-intelligence on a schedule to keep your data up to date
