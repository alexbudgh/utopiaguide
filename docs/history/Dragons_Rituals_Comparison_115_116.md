# Dragons & Rituals Comparison: Age 115 → Age 116

---

## Dragons

**General changes in Age 116:**

- **Per-dragon Cost Modifiers removed** — replaced with a flat **+15% Gold and Food** cost for any dragon started outside War
- Amethyst, Emerald, and Ruby all deal more damage on arrival and/or per tick
- Sapphire's periodic rune destruction now triggers every **3 ticks** (was every 6)

## Rituals

No changes to Rituals were announced for Age 116. The Age 115 ritual set (Ascendancy, Barrier, Benediction, Haste, Havoc, Onslaught, Stalwart) carries over unchanged — see [Age 115](Age_115.md#rituals) for the current effects.
