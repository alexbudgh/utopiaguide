# Personality Comparison: Age 115 → Age 116

Only rows that changed are shown. Personalities with no changes are omitted.

!!! note "Removed, returning, and new Personalities"
    The **Paladin** (available in Age 115) is not available in Age 116. The **Cleric**, absent since Age 113, returns in Age 116. The **Sage** is new to Age 116. See [Age 116](Age_116.md#personalities) for the full Cleric and Sage stat blocks.
