# Theme

**Theme** (or **CSS theme**) - is a game looking.

Till Age 10 - simple theme (Times New Roman font, red-blue-gray tables, big images, yellow/white text)

Age 11 - Age 14 - strong simple theme (Arial font, dark red - dark blue - dark gray tables)

Age 15 - Age 20 - pre-classic style (anime looking images)

age 21 - Age 32 - yellow classic style (3d animated images)

Age 33 - Age ?40 - white classic style (white/gray text)

Age ?41 - Age 53 - post-classic style (no images, dark red /dark blue - dark gray/transparent tables)

Since [Age of Havoc](#) players can choose 2 themes at the Account Settings/CSS theme : Neo classic theme and New Theme (now called "Default")
